## Описание программы 
Графическое приложение на Python для учёта личных расходов. Позволяет добавлять траты с указанием суммы, категории и даты, фильтровать записи, автоматически подсчитывать общую сумму за выбранный период и сохранять данные в формате JSON.


## Как использовать
- Введите сумму, выберите категорию, укажите дату (YYYY-MM-DD) → нажмите Добавить расход 
Выберите нужную категорию в выпадающем списке → Применить
Заполните поля С и По (формат YYYY-MM-DD) → Применить
Итог пересчитывается автоматически при каждом изменении таблицы
Данные сохраняются в expenses.json после каждого добавления
