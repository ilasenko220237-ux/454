## Описание
Графическое приложение для ведения личной базы данных фильмов. Позволяет добавлять фильмы, указывая название, жанр, год выпуска и рейтинг. Поддерживает фильтрацию и сохранение данных в JSON.

## Функциональные возможности
1. **Добавление фильма:** Поля: Название, Жанр, Год, Рейтинг.
2. **Валидация ввода:**
   - Год должен быть целым числом (1888–2026).
   - Рейтинг должен быть числом от 0 до 10.
3. **Просмотр:** Таблица всех добавленных фильмов.
4. **Фильтрация:** Поиск по жанру (частичное совпадение) и по точному году выпуска.
5. **Удаление:** Возможность удалить фильм из списка.
6. **Сохранение:** Данные хранятся в файле `movies.json`.
